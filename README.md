# desafio_BackEnd

## criar venv

```
python -m venv venv
```

### entrar no venv

```
.\venv\Scripts\activate
```

### install requirements

```
pip install -r requirements.txt
```

### Execute as migrações

```
python manage.py migrate
```

### Rode a aplicação

```
python manage.py runserver
```

### Rota inicial

```
http://127.0.0.1:8000/api/file/
```
